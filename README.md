# TN SkyCast - Tamil Nadu Weather App

A premium, responsive weather application specialized for the 38 districts of Tamil Nadu. Built with a focus on modern design, localization, and utility for both general users and farmers.

![App Preview](https://raw.githubusercontent.com/Aswin800/-tn-weather-app/main/preview.png) *(Add your screenshot here)*

## 🌟 Features

- **38 District Coverage**: Precise weather data for every district in Tamil Nadu.
- **Dynamic Theming**: UI transforms based on weather (Sunny, Rainy, Cloudy, Night).
- **Bilingual Support**: Full English and Tamil language integration.
- **Farmer-Centric Suggestions**: Tailored agricultural advice based on real-time climate data.
- **Interactive Search**: Smart autocomplete for quick navigation.
- **Modern Glassmorphism**: Stunning frosted-glass UI with smooth animations.
- **Weather Alerts**: Instant warnings for extreme temperatures or high winds.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Weather API**: Open-Meteo (High reliability, no key required)
- **Icons**: Lucide Icons
- **Fonts**: Inter (English), Mukta Malar (Tamil)
- **Dev Server**: Vite

## 🚀 Getting Started

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Aswin800/-tn-weather-app.git
   ```
2. Navigate to the directory:
   ```bash
   cd tn-weather-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🌾 Agriculture Logic
The app uses current temperature and humidity levels to provide advice such as:
- *Irrigation needs during high heat.*
- *Fertilizer timing relative to expected rainfall.*
- *Disease prevention during high humidity.*
